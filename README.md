# php_agenda
Exemplifica o uso de Docker, PHP + MySQL

Clonar repositório:
```
git clone https://github.com/macaroots/php_agenda.git
```

Ligar serviços com Docker Compose:
```
docker-compose up -d
```
* ```-d``` de Detach é opcional.
* Talvez seja necesário permissão de super-usuário (```sudo```).
